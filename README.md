# react app

## Requirements
- node `v12`+
- port `8080` on `localhost` must be free
- API should be running on port `3000`

## Setup
In project root directory run:  
- `npm install`

## Tests
- `npm run test` : watcher on
- `npm run test-ci` : watcher off

## Run
- `npm start`
- navigate to http://localhost:8080/
